How to find and install Netflix Games


To play, youâ€™ll have to install the game on your device first. You need:

  • A compatible device:
    • Android phone or tablet running Android 8.0 or later
    • iPhone, iPad, or iPod touch running iOS/iPadOS 15 or later
  • An active Netflix subscription
  • An internet connection to download and install the game
  • Enough storage space on your device

NOTE: Some Netflix Games have additional requirements. Check the article for a particular game for details.

From the Netflix app

The Netflix app includes a Mobile Games row on the home screen and a Games tab at the bottom.

  1. From the home screen in the app, swipe down to find the Mobile Games row, or tap the Games tab.

    2. NOTE: Not seeing the Mobile Games row or Games tab? Make sure you have the latest Netflix app version. See How to update the Netflix app on your Android device.


  2. Tap on the game, and tap Get Game. The Play Store will open.
  3. Tap Install.
    • Depending on the game and your device settings, a Permissions screen may appear. Choose Accept.
  4. The game will download and install. When complete, choose Open.
    • After a game is installed, you can open it from the Mobile Games row or Games tab in the Netflix app. You can also tap its icon on your device's home screen or in the app drawer.
  5. The game will launch. Tap the Profile icon in the game if you'd like to change to a different profile and start playing!
From the Netflix app

The Netflix app includes a Mobile Games row on the home screen.

  • From the home screen in the app, swipe down to find the Mobile Games row.

    • NOTE: Not seeing the Mobile Games row? Make sure you have the latest Netflix app version. See How to update the Netflix app on iPhone, iPad, or iPod touch.


  • Tap on the game, and then tap Get Game or Play Game.
  • Tap Get or the Cloud icon on the App Store banner.
  • The game will download and install. When complete, tap Open.
    • After a game is installed, you can open it from the Mobile Games row in the Netflix app, or by tapping its icon on your device's home screen.
  • The game will launch. Tap the Profile icon in the game if you'd like to change to a different profile and start playing!
From the App Store
  1. Open the App Store and search for the game by name or see all available Netflix games.
  2. Select the game from the search results, and tap Get or the Cloud icon.
  3. The game will download and install. When complete, tap Open.
    • After a game is installed, you can open it from the Mobile Games row in the Netflix app, or by tapping its icon on your device's home screen.
  4. If prompted to log in, tap Next and then enter your Netflix account email address and password. Select Log in.
  5. Choose your Netflix profile and start playing!

Not finding the game you're looking for? See Can't find a Netflix game.
