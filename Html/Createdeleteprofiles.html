How to create, change, or delete profiles


People who live together in a single household can have their own personalized Netflix experience. You can have up to 5 profiles on a single Netflix account.

NOTE: Profiles aren't available on devices made before 2013.

Each profile allows for its own:

  • Language preference
  • Maturity level
  • Specific viewing restrictions
  • Profile lock
  • Viewing activity log
  • Game handle
  • Game saves
  • Subtitle appearance
  • Playback settings
  • Personalized TV show and movie suggestions
  • My List
  • Ratings
  • Email

NOTE: Some profile features are not available for Kids profiles.

Profiles can be added on devices made after 2013.

  1. From a web browser, go to your Manage Profiles page. Select Add Profile.
    • In the Netflix mobile app for Android, iPhone, or iPad:
    1. From the lower right, tap My Netflix.
    2. From the upper right, tap More or .
    3. Tap Manage Profiles .
    4. Select Add Profile.
  2. Name the profile.
  3. To use the Netflix Kids experience, select Kids.
  4. Select Continue or Save. The new profile should show on the list of profiles on your account.

If you can’t create a profile from your device, visit netflix.com on a web browser and follow the steps above.

You can delete profiles on most devices.

  1. From a web browser, go to your Manage Profiles page.
    • In the Netflix mobile app for Android, iPhone, or iPad:
    1. From the lower right, tap My Netflix.
    2. From the upper right, tap More or .
    3. Tap Manage Profiles .
  2. Choose the profile you want to change.
  3. Change the name, image, or language on the profile.
    • Changes to maturity ratings must be made from your Account page.
  4. Save your changes or select Delete Profile to delete the profile entirely.

NOTE: The main profile on your Netflix account cannot be deleted. Once you delete a profile, the viewing history and game saves for that profile will no longer be available.


If you can't edit a profile from your device, visit Netflix.com on a web browser and follow the steps above.

You can customize profiles on most devices.

  1. From a web browser, go to your Manage Profiles page.
    • In the Netflix mobile app for Android, iPhone, or iPad:
    1. From the lower right, tap My Netflix.
    2. From the upper right, tap More or .
    3. Tap Manage Profiles .
  2. Choose the profile you want to change.
  3. Change the name, image, or language on the profile. If you can't edit a profile from your device, visit netflix.com on a web browser and follow the steps above.
    • Changes to maturity ratings must be made from your Account page.
  4. Save your changes.

All profile users, except for those with the Netflix Kids experience, can access and edit Profile & Parental Controls for individual profiles. Your account password is required to edit Viewing Restrictions or Profile Lock.

To edit settings for an individual profile:

  1. From a web browser, go to your Account page
  2. Select a profile name from the Profile & Parental Controls section.
  3. Select Change for the setting you want to edit for that profile.
  4. Save/Submit changes.

For more information, see Parental Controls on Netflix.

You can add a different email address to any secondary Adult profile to receive personalized recommendations and other communications from Netflix. Email addresses associated with secondary profiles can't be used to sign in to your Netflix account.

To add or change the email associated with a secondary profile:

  1. From a web browser, sign in to the profile you want to add an email for./li>
  2. Go to the Account page.
  3. Select the profile name from Profile & Parental Controls section you want to add an email for.
    • Next to Profile Email, select Add, or Change if you already added an email.
  4. Enter the desired email for the profile.
  5. Select Add Email.
    • You can select Change Email or Delete Email if you are updating an existing email.

For more information, see Parental Controls on Netflix.
